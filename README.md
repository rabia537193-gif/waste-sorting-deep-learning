# Garbage Classification using Deep Learning (CNN) ♻️📸

An automated waste management and recycling solution driven by Deep Learning. This project implements a Convolutional Neural Network (CNN) using TensorFlow and Keras to classify household waste images into 6 distinct categories, aiming to optimize automated sorting systems in smart cities.

---

## 🚀 Project Overview
Rapid urbanization requires efficient recycling methods. This project addresses the challenge by training a deep learning model to accurately identify and sort waste types, reducing human error in recycling pipelines.

### 📊 Dataset Details
The model is trained on the **Garbage Classification Dataset**, which contains images categorized into **6 classes**:
*   Cardboard 📦
*   Glass 🍾
*   Metal 🥫
*   Paper 📄
*   Plastic 🍼
*   Trash 🗑️
